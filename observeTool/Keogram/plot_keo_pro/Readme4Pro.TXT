BatchPlotKeo_2004_3axis.m:

作者：胡泽骏(huzejun@pric.gov.cn)
功能：读入keodata目录数据，设定关注时段,自动绘制每天指定的特定时间段的三波段Keogram图像，并保存到ps文件中.该程序需要调用UT4XLabel.m程序.

使用方法：
如要自动绘制从2003-12-20至2003-12-30中每天03:00:00-15:00:00UT时段内的keogram,并自动保存至ps文件中.
1.在matlab环境下运行BatchPlotKeo_2004_3axis.m；
2.在弹出的“Open ASI MAT file”对话框中，选择所关注日期时段的.mat(如2003-12-20至2003-12-30日)所在的目录(keoData_200311_200403)中的任意文件；
3.在matlab命令行显示的"Please input the start day [yyyy-mm-dd]："后输入关注日期段的开始日期，如：2003-12-20;
4.在"Please input the end day [yyyy-mm-dd]:"后输入关注日期时段的结束日期，如：2003-12-30；
5.在"Please input the start time on every day[hh:mm:ss]:"后输入关注时段的开始时间，如：03:00:00;
6.在"Please input the end time on every day[hh:mm:ss]:"后输入关注时段的结束时间，如：15:00:00.
7.程序自动把2003-12-20至2003-12-30中每天03-15UT时段的keogram绘制在同一个ps文件中.

注意：
1. 如果只想观测一天某一时段，只用开始日期和结束日期输入相同日期即可；
2. 如果想知道2003-12-20/10:00:00-2003-12-21/10:00:00 UT连续24小时的极光keogram，其输入方法为：
	"Please input the start day [yyyy-mm-dd]：" 2003-12-20
	"Please input the end day [yyyy-mm-dd]:"2003-12-20
	"Please input the start time on every day[hh:mm:ss]:"10:00:00
	"Please input the end time on every day[hh:mm:ss]:"34:00:00 (说明:2003-12-21/10:00:00UT相当于2003-12-20/34:00:00UT)
3. 使用中如果出现bug，请及时反馈给作者，以便更加方便大家的使用.
